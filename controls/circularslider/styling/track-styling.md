# .NET MAUI CircularSlider Track Styling

The CircularSlider for .NET MAUI provides the following styling options for the backtrack (the arc the thumb slides along):

* `BackTrackThickness` (`double`)&mdash;Specifies the thickness of the CircularSlider's backtrack.
* `BackTrackColor` (`Color`)&mdash;Defines the color of the CircularSlider's backtrack.
* `BackTrackStyle` (`Style`)&mdash;Defines a custom style for the CircularSlider's backtrack.

The CircularSlider for .NET MAUI provides the following styling options for the range track (the track between `OriginValue` and `Value`):

* `RangeTrackFill` (`Color`)&mdash;Defines the fill color for the CircularSlider's range track.
* `RangeTrackStyle` (`Style`)&mdash;Defines a custom style for the CircularSlider's range track.
