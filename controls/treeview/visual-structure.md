# .NET MAUI TreeView Visual Structure

The visual structure of the .NET MAUI TreeView represents the anatomy of the UI component. Being familiar with the visual elements of the TreeView allows you to quickly find the information required to configure them.

The following image shows the anatomy of the TreeView.

![TreeView Visual Structure](images/treeview-visual-structure.png "Visual elements of TreeView")

## Displayed Elements

- **TreeView Item**&mdash;On mobile devices, the user can expand the item by tapping on it. While on desktop devices, the user has to click the **>** arrow.
- **TreeView Selected Item**&mdash;Represents the item currently selected within the TreeView.
- **(Desktop) Hovered Item**&mdash;Represents the item currently hovered by the mouse pointer.
- **Expand/Collapse Indicator**&mdash;The indicator used for expanding/collapsing an item. On desktop devices, the user clicks the **>** arrow to expand an item. By default, on mobile, the user can tap on the item to expand it.
- **CheckBox element**&mdash;Represents a Checkbox element in the TreeView item.
