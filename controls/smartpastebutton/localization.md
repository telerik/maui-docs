# SmartPasteButton Localization

The SmartPasteButton for .NET MAUI provides language localization. The localization keys are described in the tables below:

>caption SmartPasteButton localization keys

| Localization Key | Default Value |
| -----------------| ------------- |
| `SmartPasteButton_Text`  | `Smart Paste` |
| `SmartPasteButton_CancelText`  | `Cancel` |


> important To learn in details about the localization process of Telerik UI for .NET MAUI components, please go through the common [Localization and Globalization]({%slug globalization-localization%}) topic.
