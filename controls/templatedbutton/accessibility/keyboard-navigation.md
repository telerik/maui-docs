# .NET MAUI TemplatedButton Keyboard Navigation Support on Desktop

The [Telerik UI for .NET MAUI TemplatedButton]({%slug templatedbutton-overview %}) provides keyboard navigation support on `WinUI` and `MacCatalyst`.

| Hotkey | Action |
| ------ | ------ |
| `Tab` | Enters or exits the TemplatedButton. |
| `Enter` | Makes the TemplatedButton perform a click action. |
| `Space` | Makes the TemplatedButton perform a click action. |
