# .NET MAUI SideDrawer Visual Structure

The visual structure of the .NET MAUI SideDrawer represents the anatomy of the UI component. Being familiar with the visual elements of the SideDrawer allows you to quickly find the information required to configure them.

The following image shows the anatomy of the SideDrawer.

![SideDrawer Visual Structure](images/sidedrawer-visual-structure.png)

## Displayed Elements

* **Main Content**&mdash;The primary content area of the application that is visible when the SideDrawer is closed.
* **Drawer Content**&mdash;The content area that slides in from the side of the screen when the SideDrawer is opened. This area typically contains navigation options or additional information.
