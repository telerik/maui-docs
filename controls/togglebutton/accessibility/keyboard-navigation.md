# .NET MAUI ToggleButton Keyboard Navigation Support on Desktop

The [Telerik UI for .NET MAUI ToggleButton]({%slug togglebutton-overview %}) provides keyboard navigation support on `WinUI` and `MacCatalyst`.

| Hotkey | Action |
| ------ | ------ |
| `Tab` | Enters or exits the ToggleButton. |
| `Enter` | Makes the ToggleButton perform a click action to trigger the toggled and untoggled states. |
