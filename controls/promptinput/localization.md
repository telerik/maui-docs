# PromptInput Localization

The PromptInput for .NET MAUI provides language localization. The localization keys are described in the tables below:

>caption TimePicker localization keys

| Localization Key | Default Value |
| -----------------| ------------- |
| `PromptInput_PickFile`  | `Attach files` |
| `PromptInput_PickPhoto`  | `Attach photos` |
| `PromptInput_TakePhoto`  | `Take photo` |
| `PromptInput_TextInputPlaceholder`  | `Type a message...` |

> important To learn in details about the localization process of Telerik UI for .NET MAUI components, please go through the common [Localization and Globalization]({%slug globalization-localization%}) topic.
